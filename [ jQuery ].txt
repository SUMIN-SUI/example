1. jQuery 개요
	- javascript 라이브러리 중 하나 ( vue.js, react.js ... )
	- legacy 프로젝트에서는 아직도 선호되는 기술 중 하나
	- javascript : MS --> 개발자 --> ECMA에서 js 표준을 제정 ECMAScript
	  javascript v5 이후가 표준 ES6
	
	- DOM에 접근해서 동적처리를 하는 js가 개발하기 어렵고, 표준도 안되는 문제 , , ,
	  ==> 그런 관계로 jQuery 언어가 생겼다.
	- "존 레식"이 2006년에 만들었다.

2. 특징
	- 모든 처리를 함수로 한다.
	document.getElementById( 'abc' ).innerHTML = 값; // 속성
	$( 'abc' ).html( '값' )
	
	- CSS 선택자와 동일한 선택자를 사용한다.

	- jQuery 라이브러리를 먼저 코드 내에서 삽입한 후 사용
	 url : jQuery.com

	- 라이브러리를 사용하려면
		1) 직접 다운
		2) cdn 방식 - 반드시 인터넷이 되는 상태에서만 사용 가능

	- jQuery 사용은 모두 함수로만!!
	- 시각적 효과를 쉽게
